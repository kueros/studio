# **App Name**: SwiftLogistics Demo

## Core Features:

- Header Component: Sticky navigation bar with logo, links (Home, Services, About, Contact), and a 'Quote Now' button. Includes a functional hamburger menu for mobile devices.
- Hero Section: Full-screen hero section with a looping, muted background video. Dark overlay for text readability. Includes a title, subtitle, and a call-to-action button linking to the services section. Supports local video upload.
- Services Section: Responsive grid displaying services offered (e.g., Land Transport, Storage, Distribution). Each service includes an icon, title, and description with a hover effect.
- About Us Section: Two-column layout for 'About Us' section. One column with a placeholder image, the other with a title, company description, and a list of key values.
- Contact Section: Two-column layout for 'Contact' section. One column with a contact form (Name, Email, Phone, Message). The other shows contact info (Address, Phone, Email, Hours) with icons and a placeholder for an interactive map.
- Footer Component: Dark-background footer with company logo/name, quick links, basic contact info, social media icons, and a copyright notice.
- Floating WhatsApp Button: Fixed floating WhatsApp button in the bottom-right corner, linked to a predefined WhatsApp number.

## Style Guidelines:

- Primary color: White or light gray for a clean background.
- Secondary color: Dark blue or gray for text and important elements.
- Accent: Teal (#008080) for buttons, highlights, and the WhatsApp button to draw attention.
- Use Inter font for a modern and readable look.
- Font Awesome icons for services, contact information, and social media links.
- Ensure a responsive layout that adapts to different screen sizes, using Tailwind CSS grid and flexbox utilities.
- Smooth scroll for internal navigation links to provide a better user experience.

## Original User Request:
Crea una página web de demostración de una sola página para una empresa de logística, utilizando HTML y Tailwind CSS. El diseño debe ser moderno, limpio y responsivo, similar en estructura y estilo a https://www.bk-logistica.com.ar/, pero con contenido genérico de marcador de posición.

La página debe incluir las siguientes secciones:

1.  **Header:** Barra de navegación fija (sticky) en la parte superior con un logo a la izquierda, enlaces de navegación (Inicio, Servicios, Nosotros, Contacto) y un botón "Cotizar Ahora" a la derecha. Debe incluir un menú hamburguesa funcional para dispositivos móviles.
2.  **Sección Hero:** 
Una sección introductoria a pantalla completa que utilice un video de fondo en bucle y silenciado. El prompt debe permitir subir o seleccionar un archivo de video local (por ejemplo, desde la carpeta "Descargas") para usarlo como fondo. Sobre el video, debe haber una superposición oscura para mejorar la legibilidad del texto. Incluir un título principal llamativo (ej: "Soluciones Logísticas Integrales"), un subtítulo descriptivo y un botón de llamada a la acción que dirija a la sección de servicios.

3.  **Sección Servicios:** Título de la sección y una cuadrícula (grid) responsiva que muestre varios servicios ofrecidos (ej: Transporte Terrestre, Almacenamiento, Distribución, etc.). Cada servicio debe tener un icono (usar Font Awesome), un título y una breve descripción. Utilizar tarjetas con un ligero efecto hover.
4.  **Sección Nosotros:** Diseño de dos columnas en pantallas grandes. Una columna con una imagen (usar placeholder) y la otra con un título, párrafos descriptivos sobre la empresa y una lista de puntos clave o valores.
5.  **Sección Contacto:** Título de la sección y un diseño de dos columnas. Una columna con un formulario de contacto (campos para Nombre, Email, Teléfono, Mensaje y botón de envío). La otra columna debe mostrar información de contacto (Dirección, Teléfono, Email, Horario) con iconos asociados y un espacio reservado (placeholder) para un mapa interactivo.
6.  **Footer:** Sección de pie de página con fondo oscuro. Debe incluir el logo o nombre de la empresa, enlaces rápidos, información de contacto básica, iconos de redes sociales y un aviso de derechos de autor.
7.  **Botón Flotante:** Un botón flotante fijo en la esquina inferior derecha con el icono de WhatsApp, enlazado a un número de WhatsApp predefinido.

**Requisitos Técnicos y de Estilo:**

* Utilizar **HTML5** semántico.
* Estilar exclusivamente con **Tailwind CSS**.
* Incluir **Font Awesome** para los iconos.
* Utilizar la fuente **Inter** de Google Fonts.
* Asegurar que el diseño sea **totalmente responsivo** y se vea bien en móviles, tabletas y escritorios.
* Implementar **scroll suave** (smooth scroll) para los enlaces internos de navegación.
* Usar imágenes de marcador de posición (ej: de placehold.co) y textos genéricos que puedan ser fácilmente reemplazados.
* El código debe estar bien comentado.
