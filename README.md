# Studio Next.js Architecture 🚀

Este proyecto es una muestra de arquitectura moderna de frontend utilizando **Next.js 15** y **TypeScript**. Está diseñado como una landing page institucional de alto rendimiento, con un enfoque en la modularidad de componentes y la optimización de la experiencia de usuario (UX).

## 🛠️ Stack Tecnológico

- **Framework:** Next.js 15 (App Router).
- **Lenguaje:** TypeScript (Tipado estricto para mayor robustez).
- **Estilos:** Tailwind CSS.
- **Componentes de UI:** Shadcn/UI (basado en Radix UI).
- **Animaciones & Interactividad:** Lucide React para iconografía y componentes reactivos de alta performance.

## 🌟 Características Destacadas

- **Component-Driven Development:** Arquitectura basada en componentes reutilizables y desacoplados.
- **Responsive Design:** Optimizado para dispositivos móviles utilizando hooks personalizados como `use-mobile`.
- **AI-Ready Structure:** Incluye una capa de integración preparada para servicios de Inteligencia Artificial (instancias de AI pre-configuradas).
- **Modern UI Patterns:** Implementación de modales, menús laterales (sheets), y secciones dinámicas con estados de React.

## 📁 Estructura del Proyecto

- `src/app/`: Rutas y layouts principales del sistema.
- `src/components/`: Biblioteca de componentes de negocio (Hero, Services, Contact).
- `src/components/ui/`: Componentes base de alta fidelidad (Shadcn/UI).
- `src/ai/`: Capa de lógica para futuras integraciones de IA.
